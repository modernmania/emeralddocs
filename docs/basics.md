## 3. Syntax Basics

- Line-oriented statements.
- Blocks use braces `{ ... }`.
- Comments start with `#` or `--`.
- Strings use single quotes `'text'` or double quotes `"text"`.
- Identifiers: letters/underscore first, then letters/digits/underscore.

Example:

```emer
# this is a comment
var name = "Emerald"
print(name)
```
